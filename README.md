# ESG Quantitative Screening Dashboard

## Overview
A category-relative, fully-automated ESG compliance platform for RIAs and asset managers. Enables quarterly screening, portfolio documentation, and investment policy compliance with an “Elite or Replace” standard.

## Features
- Category-relative percentile ESG ranking—no more “utilities vs. tech” false comparisons
- Automated IC-ready PDF reports for each category
- “Elite or Replace” compliance model with only ≤25th percentile allowed as standard
- Visual, auditable documentation for SEC/DOL/EU compliance
- “Methodology” tab with full rationale, thresholds, and weights

## Installation
- Python 3.9+ required
- Install dependencies: `pip install -r requirements.txt`
- Launch with: `streamlit run app.py`

## Usage
- Export YCharts ESG data and model portfolio as CSVs
- Upload ESG universe and model to dashboard
- Review, filter, and export category PDFs for compliance
- Use Methodology tab for audit support or client education

## File/Folder Descriptions
- `app.py` – Main dashboard app (Streamlit)
- `config.py` – Thresholds and metric weights
- `report_generator.py` – PDF compliance report logic
- `requirements.txt` – All dependencies
- `README.md` – Project documentation, origin statement

## Customization
- Change `config.py` to update thresholds or metric weights
- Add/remove screening metrics by editing `config.py` and dashboard
- Understand .csv upload templates before using
